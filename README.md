#  Teste Prático - Ellos Design

Projeto desenvolvido por **Grazzielly Pereira** como parte do processo seletivo para vaga de estágio **Frontend / Full-Stack**.


##  Sobre o projeto

A proposta deste teste foi desenvolver uma **landing page responsiva**, seguindo a estrutura base fornecida, mas com liberdade criativa para definir identidade visual, layout e experiência do usuário.

Neste projeto, desenvolvi uma página para uma marca fictícia chamada **Pulse Web Studio**, focada em criação de sites e soluções digitais para empresas.


##  Objetivos do desenvolvimento

Durante a construção do projeto, busquei aplicar:

- Organização e clareza no código
- Responsividade (mobile-first)
- Boa experiência do usuário (UX)
- Interface moderna e atrativa (UI)
- Integração entre frontend e backend
- Validação de dados no formulário
- Boas práticas de desenvolvimento

##  Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **PHP**
- **MySQL**


##  Funcionalidades

- Landing page responsiva
- Seção de apresentação (hero)
- Formulário funcional com:
  - Validação de campos
  - Máscara de telefone
  - Feedback visual (loading + mensagem)
- Animações ao scroll
- Cards de benefícios com ícones
- FAQ interativo
- Integração com backend em PHP
- Armazenamento de dados no banco MySQL
