#  Teste Prático - Ellos Design

Projeto desenvolvido por **Grazzielly Pereira** como parte do processo seletivo para vaga de estágio **Frontend / Full-Stack**.

---

##  Sobre o projeto

A proposta deste teste foi desenvolver uma **landing page responsiva**, com liberdade criativa para identidade visual, layout e experiência do usuário.

O projeto foi construído para uma marca fictícia chamada **Pulse Web Studio**, com foco em apresentação de serviços digitais e captação de leads.

##  Objetivos do desenvolvimento

Durante o desenvolvimento, busquei aplicar:

- Organização e legibilidade de código
- Responsividade (mobile-first)
- Interface moderna e intuitiva
- Boa experiência do usuário (UX)
- Integração entre frontend e backend
- Validação de dados no formulário


##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP
- MySQL
- Firebase Hosting (deploy do frontend)



##  Funcionalidades

- Landing page responsiva
- Hero com animações de fundo
- Seção institucional
- Cards de benefícios com ícones
- FAQ interativo
- Máscara de telefone
- Botão com estado de carregamento
- Validação de formulário
- Integração com backend (PHP)
- Armazenamento de dados no MySQL
